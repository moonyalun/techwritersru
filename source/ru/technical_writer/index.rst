Технический писатель
====================

Технический писатель - это специалист, который создает техническую документацию для программного обеспечения или других технических продуктов. Технический писатель предоставляет сложную информацию в понятной и доступной форме для :ref:`целевой аудитории <users_documentation_roles>`.
